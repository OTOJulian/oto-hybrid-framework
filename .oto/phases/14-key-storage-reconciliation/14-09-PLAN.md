Sync hygiene: settings-integrations.md is GSD-shared — keep diffs minimal and scoped to the defect lines.

Portability note (checker info issue): the workflow's bash blocks run under macOS's default bash 3.2, and an EMPTY array expanded as `"${WS_ARGS[@]}"` is an "unbound variable" error there whenever `set -u` is in effect. All WS_ARGS expansions therefore use the guarded form `${WS_ARGS[@]+"${WS_ARGS[@]}"}` — expands to nothing when the array is empty, to the quoted elements otherwise, and is safe under bash 3.2 + `set -u`.
