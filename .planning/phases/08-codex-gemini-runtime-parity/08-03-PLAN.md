| Boundary | Description |
|----------|-------------|
| Repo source ↔ runtime config dir | Codex install copies + emits files into `~/.codex/`. Untrusted upstream content could embed identity strings or hooks if not sanitized. |
| `~/.oto/defaults.json` ↔ TOML emit | User-controlled JSON values flow into per-agent .toml strings. Parse-confusion could embed untrusted content. |
| `runtime-codex.cjs::agentSandboxes` ↔ Codex agent router | The map controls per-agent file-system access. Wrong sandbox_mode = wrong privilege. |
| install.cjs orchestrator ↔ adapter.emitDerivedFiles | Hook receives `{ configDir, repoRoot, otoVersion, projectRoot }`; wrong values could redirect emit to attacker-controlled path. |
