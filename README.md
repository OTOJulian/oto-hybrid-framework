# oto

oto is a personal hybrid AI-CLI framework that combines GSD's planning and execution workflow with selected Superpowers capabilities behind one consistent `/oto-*` command surface across Claude Code, Codex, and Gemini CLI.

## Install

```sh
npm install -g https://github.com/OTOJulian/oto-hybrid-framework/archive/vX.Y.Z.tar.gz
```

Replace `vX.Y.Z` with a release tag when one is available.

## Status

alpha - Phase 2 ships the rebrand engine; Phase 3 wires the real installer.
